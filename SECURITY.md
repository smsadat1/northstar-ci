# Security Policy

Northstar CI is currently in pre-release.

If you discover a security vulnerability, please do not open a public issue.

Instead, contact:
smsadat788@gmail.com

or open a private disclosure through GitHub Security Advisories.

The project is under active development and security boundaries may change between releases.

## Out of Scope

Northstar executes user-provided workloads.

Execution of arbitrary commands inside the intended execution environment is not considered a security vulnerability by itself.

Security issues are limited to:
- sandbox escapes
- privilege escalation
- unauthorized runner access
- authentication bypasses
- data disclosure
